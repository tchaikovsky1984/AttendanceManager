# AttendanceManager
A basic non-GUI attendance checking and absence managing program written in C.

## Usage
Use a C compiler such as GCC or Clang to compile the C file and run it

Linux:
```bash
# Compiling C file
cc att.c
# Running executable file
./a.out
```

Windows:
```
:: Compiling C file
gcc -o att.exe att.c
:: Running Executable file
att.exe
```
